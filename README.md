# OSINT-Portfolio

Hi, I’m Sabrine Quddumi, an aspiring OSINT Analyst.  
This portfolio showcases my investigations and reports using open-source intelligence (OSINT) techniques. Each project demonstrates research, digital footprint analysis, geolocation, threat intelligence, social network mapping, and case simulation.

---

## Projects

### 1️⃣ Digital Footprint Investigation
**Folder:** [01_DigitalFootprint](01_DigitalFootprint/)  
**Description:** Investigated the online presence of a persona, analyzed social media accounts, email breaches, and digital traces using OSINT tools.  
**Tools Used:** WhatsMyName, HaveIBeenPwned, Google Reverse Image Search, EXIF.tools  

### 2️⃣ Geolocation Challenge
**Folder:** [02_Geolocation](02_Geolocation/)  
**Description:** Used geolocation techniques to determine locations from images, shadows, and map data.  
**Tools Used:** Google Maps, Google Earth, OpenStreetMap, SunCalc  

### 3️⃣ Threat Intelligence Report
**Folder:** [03_ThreatIntel](03_ThreatIntel/)  
**Description:** Investigated domains, emails, and potential cyber threats using open-source intelligence.  
**Tools Used:** Shodan, VirusTotal, URLscan.io, Whois Lookup  

### 4️⃣ Social Network Mapping
**Folder:** [04_SocialNetworkMapping](04_SocialNetworkMapping/)  
**Description:** Analyzed connections between social media accounts to map networks and relationships.  
**Tools Used:** Maltego CE, Gephi, LinkedIn, Twitter  

### 5️⃣ Case Simulation
**Folder:** [05_CaseSimulation](05_CaseSimulation/)  
**Description:** Simulated a real-world OSINT investigation, combining multiple techniques to gather actionable intelligence.  
**Tools Used:** WhatsMyName, Google, Open-source datasets, Visualization tools
